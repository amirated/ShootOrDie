# Shoot or Die

Shoot or Die is a 2d, single player shooting game.

## Setup

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install pygame.

```bash
pip3 install pygame
```

## Run

```bash
python3 main.py
```
